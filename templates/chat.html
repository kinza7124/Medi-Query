About Medical AI Chatbot

What is this?

This is an AI-powered Medical Chatbot designed to provide informational support for health-related questions. It uses advanced AI technology combined with a verified medical knowledge base to deliver accurate, context-aware responses.

Knowledge Base

  • Source: Medical reference documents (Medical_book.pdf)
  • Topics Covered: General health, symptoms, conditions, treatments, medications, and wellness advice
  • Technology: RAG (Retrieval-Augmented Generation) pipeline with semantic search
  • Embeddings: all-MiniLM-L6-v2 (384-dimensional vectors)

How to Use

  1. Ask Questions: Type your health-related query in the input field
  2. Be Specific: Provide clear details for better responses (symptoms, duration, etc.)
  3. Follow-up: Ask clarifying questions - the bot remembers conversation context
  4. Clear Chat: Use the trash icon to start a new conversation
  5. Theme Toggle: Switch between light and dark mode with the moon/sun icon

Important Disclaimer

This AI provides informational content ONLY.

  • Always consult qualified healthcare professionals for medical diagnosis
  • Do not use for emergency situations - call emergency services instead
  • Information may not be complete or up-to-date
  • Not a substitute for professional medical advice, diagnosis, or treatment

Privacy & Security

  • Conversations are processed securely using Groq AI infrastructure
  • Session-based memory - chats are not permanently stored
  • Vector database: Pinecone (secure cloud storage)
  • No personal medical data is shared with third parties

Technical Features

  • Query Rewriting: AI understands pronouns and context from chat history
  • Smart Retrieval: MMR + Cross-Encoder reranking for best context
  • Response Quality: Faithful to medical sources with strict abstention
  • Model: Llama 3.3 70B (high-quality medical reasoning)
