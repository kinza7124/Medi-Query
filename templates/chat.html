About MediQuery

What is MediQuery?

MediQuery is an AI-powered Medical Chatbot that answers health questions using a verified medical knowledge base. It uses a Retrieval-Augmented Generation (RAG) pipeline — retrieving relevant passages from medical documents before generating a grounded, evidence-based response.

Knowledge Base

  • Source: Gale Encyclopedia of Medicine (Medical_book.pdf)
  • Topics: Diseases, symptoms, causes, treatments, prevention, complications, diagnostics
  • Chunking: Hybrid token-aware chunking — section metadata + semantic sentence boundaries (avg ~74 tokens/chunk)
  • Embeddings: BAAI/bge-small-en-v1.5 (384-dim, retrieval-optimised)
  • Vector DB: Pinecone (8,845 indexed chunks)

Retrieval Pipeline

  • Stage 1 — Recall: Hybrid BM25 (exact term matching) + MMR dense retrieval (semantic diversity), weighted 40/60
  • Stage 2 — Precision: Cross-Encoder reranker (ms-marco-MiniLM-L-6-v2) keeps top 8 most relevant chunks
  • Metadata filtering: Query intent is detected (symptoms / causes / treatment / diagnosis / prevention / complications) and Pinecone is filtered to matching sections before retrieval
  • Query rewriting: llama-3.1-8b-instant rewrites your query for optimal search — resolves pronouns, fixes typos, removes filler words

AI Models

  • Answer generation: Llama 3.3 70B Versatile (via Groq) — high-quality medical reasoning
  • Query rewriting: Llama 3.1 8B Instant (via Groq) — fast, lightweight query optimisation
  • Reranker: cross-encoder/ms-marco-MiniLM-L-6-v2 (local CPU inference)
  • Embeddings: BAAI/bge-small-en-v1.5 (local CPU inference)

How to Use

  1. Ask naturally: Type any health question — typos and informal phrasing are handled automatically
  2. Be specific: Mention the condition, symptom, or topic for more focused answers
  3. Follow up: The bot remembers the last 3 exchanges for pronoun resolution and context
  4. New chat: Click "New conversation" in the sidebar or the trash icon to reset
  5. Dark mode: Toggle with the moon/sun icon in the top bar

Important Disclaimer

This AI provides informational content ONLY.

  • Always consult a qualified healthcare professional for diagnosis and treatment
  • For emergencies, call 911 or your local emergency number immediately
  • Responses are grounded in the indexed knowledge base — information may not cover all conditions or latest guidelines
  • Not a substitute for professional medical advice, diagnosis, or treatment

Privacy & Security

  • Conversations are processed via Groq's secure API infrastructure
  • Session memory is stored in your browser session only — not persisted to any database
  • History is capped at 20 messages per session to prevent data accumulation
  • No personal medical data is stored or shared with third parties
